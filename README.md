Ho conseguito una **Laurea Magistrale in Business Informatics & Data Science** presso l'Università di Camerino, costruendo competenze che spaziano dallo sviluppo software tradizionale fino all'intelligenza artificiale applicata, con una particolare focalizzazione su:
